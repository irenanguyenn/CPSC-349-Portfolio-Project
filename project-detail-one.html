FTP Server and Client

Description: The FTP Server and Client was designed to showcase our understanding of networking protocols, socket programming, and the architecture between server and client for class CPSC 471: Computer Communications. The system demonstrates essential FTP functionalities, including uploading, downloading, and managing files between client and server.

Technologies Used:

  • Programming Language: Python
  • Core Libraries: socket

Features:

  • File Uploads and Downloads
  • File Directory Management
  • Error Handling

Challenges:

  • Ensuring data integrity was our priority as file transfers needed to be accurate, with no data loss or corruption. To prevent such incident to occur, checksum verification was implemented.
  • Implementing FTP protocol standards was difficult as we had little understanding of the commands while adhering to protocol standards, but after following RFC documentation for FTP protocols and creating modular code for command handling, implementation was a breeze.

Outcome: The project successfully demonstrated a functional FTP system, providing a hands-on understanding of client-server communication and network programming. It served as an excellent learning experience for real-world implementation of networking concepts.
